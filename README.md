## 🎯 核心项目详解

### 🏆 Assignment 1 - 系统管理程序

#### Q3: VMIM系统 - 自动售货机库存管理
一个完整的命令行库存管理系统，实现了：
- ✅ 添加零食及数量 (`ADD <snack> <quantity>`)
- ✅ 删除指定索引的零食 (`REMOVE <index>`)
- ✅ 打印当前库存 (`PRINT`)
- ✅ 完善的输入验证和错误处理

**运行方式：**
```bash
cd assignment1
python Q3.py
```

#### Q4: 悉尼大学课程注册系统
模拟大学课程注册系统，包含学生和课程管理：
- 👨‍🎓 学生注册和退课功能
- 📚 查询学生课程表
- 📊 课程容量管理
- 🎓 自动分配辅导班（Tutorial）

**特色功能：**
- 根据学生ID自动分配辅导班组
- 课程满员检测
- 完整的命令行交互界面

### 🏆 Assignment 2 - 模拟器项目

#### Q1: 日本武士刀重铸模拟器
读取武士刀蓝图文件，计算与完美武士刀的相似度：
- 🗡️ 武士刀部件对比算法
- 📊 相似度评分系统
- 🔄 重铸建议生成

#### Q2: 澳大利亚动物迁移模拟
模拟动物在不同州之间的安全迁移：
- 🦘 读取CSV格式的动物数据
- 🗺️ 计算最安全的迁移路径
- 📈 迁移风险评估

#### Q3: 约旦铁路建设模拟器
优化铁路站点建设成本：
- 🚂 基于学生ID的复杂成本计算
- 💰 成本优化算法
- 📍 站点位置规划

## 📝 代码规范

本仓库中的所有代码遵循以下规范：

1. **详细的中文注释**：每个文件都包含：
   - 文件头部的功能说明
   - 函数的文档字符串（docstring）
   - 关键代码行的解释性注释
   - 复杂算法的逻辑说明

2. **清晰的命名规范**：
   - 变量名具有描述性
   - 函数名反映其功能
   - 类名使用驼峰命名法

3. **模块化设计**：
   - 功能分离，职责单一

4. **错误处理**：
